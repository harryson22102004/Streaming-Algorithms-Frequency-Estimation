# Data Stream Frequency Estimation
Implementing space-efficient algorithms for real-time analysis of massive data streams.

### 🔬 Research Highlights
- **Count-Min Sketch**: Uses hashing to provide $(\epsilon, \delta)$-approximations of element frequencies using sub-linear space.
- **Morris Counter**: Implements approximate counting to track large numbers using logarithmic bit space.
- **Flajolet-Martin Algorithm**: Provides cardinality estimation for distinct elements in a stream.

**Tech Stack:** Python, NumPy.
